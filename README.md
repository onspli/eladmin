# eladmin
Eladmin generates CRUD admin interface for Eloquent. [Docs and examples](https://onspli.github.io/eladmin/)

## Table of Contents

- [Installation](#installation)
- [Contacts](#contacts)

## Installation
You can install Eladmin using composer:
```
composer require onspli/eladmin
```
